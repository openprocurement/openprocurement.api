Period
======

Schema
------

:startDate:
    string, :ref:`date`

    |ocdsDescription|
    The start date for the period.

:endDate:
    string, :ref:`date`

    |ocdsDescription|
    The end date for the period.

`startDate` should always precede `endDate`.
